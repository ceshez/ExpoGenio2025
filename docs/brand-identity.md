# Genio Brand Identity

## Core tagline

Genio: Tu web, tu manera.

This phrase should guide the product, landing page, editor experience, and visual identity. Genio should communicate personal control, creative freedom, and simplicity for people who want to build a professional website without needing technical knowledge.

## Name concept

The name Genio is built around three ideas:

```txt
GEN = generar / generative
I = interactivo / interactive
O = objetos / objects
```

This means Genio should feel like a product that helps users generate, interact with, and arrange visual objects to create their own website.

## Logo concept

The Genio logo is made from simple geometric shapes that form the letter `G`.

The logo concept includes:

- A play/start shape that represents action, beginning, and launching something new.
- A slider/control element that represents customization, personal control, and creative editing.
- Modular geometric pieces that represent blocks, objects, sections, and drag-and-drop website creation.

The visual identity should use this logic beyond the logo. Genio should feel modular, editable, interactive, and geometric without becoming cold or overly technical.

## Brand personality

Genio should feel:

- Friendly.
- Warm.
- Trustworthy.
- Creative.
- Modular.
- Interactive.
- Professional.
- Empowering.
- Easy to use.
- Business-ready.

Genio should not feel:

- Generic.
- Overly corporate.
- Cold.
- Developer-only.
- Childish.
- Overly pink or stereotypically feminine.
- Like a beauty, fashion, or lifestyle brand.
- Like a generic startup template.

## Target audience

Genio is made for entrepreneurs, small businesses, local stores, freelancers, service providers, and Costa Rican / Latin American SMEs.

A significant part of this audience is women-owned SMEs. The product should feel especially welcoming, elegant, approachable, empowering, and supportive for women entrepreneurs, without using stereotypes or making the interface feel childish or decorative.

The visual tone should say:

```txt
You can create a professional website for your business, your way, without coding.
```

## Brand colors

Core logo palette:

- Deep blue: `#2E388E`
- Purple: `#5C2483`
- Bright blue: `#257DC1`

Supporting product palette:

- Turquoise accent: `#14B8A6`
- Soft background: `#F8FAFC`
- White: `#FFFFFF`
- Dark text: `#0F172A`
- Muted text: `#64748B`
- Soft border: `#E2E8F0`

Optional restrained warmth:

- Soft cream.
- Warm beige.
- Very light peach.
- Soft lavender-tinted neutrals.
- Subtle coral highlights.

Warm accents should support the blue/purple/bright-blue identity, not replace it.

## Color usage principles

- Use deep blue as the primary trust and brand anchor.
- Use bright blue for interactive emphasis, hover states, links, and active moments.
- Use purple for distinctive brand moments, gradients, and geometric identity details.
- Use turquoise as a restrained accent for freshness, progress, and positive action.
- Use warm neutral accents to make landing pages and marketing surfaces feel more human.
- Avoid making the product visually dominated by pink, purple, or loud gradients.

## Typography

The brand typography direction should be inspired by Garet as the primary brand typeface.

Use rounded, geometric, friendly sans-serif typography for:

- Hero headlines.
- Main section headings.
- Brand moments.
- Landing page titles.

Use readable Inter-style typography for:

- Body copy.
- UI labels.
- Forms.
- Dashboard text.
- Product descriptions.

References such as Unio and Nifty can inspire the rounded geometric feel, but the product should remain readable and professional.

## Visual motifs

Use visual details inspired by the logo:

- Modular blocks.
- Geometric cards.
- Play/start shapes.
- Slider/control details.
- Layered panels.
- Editable website section mockups.
- Drag-and-drop object patterns.
- Subtle geometric backgrounds.

These motifs should feel connected to website creation and personalization. They should not become random decoration.

## Landing page direction

The Genio landing page should feel:

- Warm and welcoming.
- Professional for real small businesses.
- More distinctive than a generic SaaS page.
- Clear about what the product does.
- Focused on reassurance and progress.

The narrative should help the user feel:

1. This is for my business.
2. I can create a website without coding.
3. I can personalize it my way.
4. I can publish and grow with Genio.

Recommended copy directions:

- `Crea tu web, a tu manera.`
- `Crea la página web de tu negocio sin complicarte.`
- `Tu negocio merece verse profesional.`
- `Empieza con una plantilla y personalízala a tu ritmo.`
- `Sin código, sin complicarte, a tu manera.`

Avoid:

- Fake statistics.
- Fake testimonials.
- Generic startup buzzwords.
- Vague superlatives.
- Overpromising unfinished features.
- Overly technical language.

## UI and product direction

The product UI should communicate that building a site is visual, guided, and manageable.

Important UI concepts:

- Blocks.
- Objects.
- Sections.
- Controls.
- Customization.
- Preview.
- Publish.

Use clear Spanish labels. Avoid developer terms in user-facing UI unless the screen is intentionally technical.

## Rule of thumb

Every Genio visual decision should support this idea:

```txt
Genio gives small business owners control, creativity, and confidence to create their website their way.
```
